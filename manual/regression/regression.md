# Regression Tests

## Корзина
### TC-REGRESSION-01: Добавление товара в корзину авторизованным пользователем
**Описание:** Проверка добавления товара в корзину.  
**Предусловие:** Пользователь авторизован в системе.  
**Шаги:**
1. Выбрать любую категорию, например "Computers".
2. Выбрать любой из товаров категории и нажать кнопку "add to cart".
3. Открыть корзину.  

**Expected result:**  
- Корзина открыта, добавленный товар из категории книги отображается в корзине.  



### TC-REGRESSION-02: Удаление товара из корзины авторизованным пользователем
**Описание:** Проверка удаления товара из корзины.  
**Предусловие:** Пользователь авторизован в системе.  
**Шаги:**
1. Выбрать любую категорию, например "Books".
2. Выбрать любой из товаров категории и нажать кнопку "add to cart".
3. Открыть корзину.
4. Нажать кнопку "remove from cart".  

**Expected result:**  
- Корзина открыта, товар удален из корзины, корзина пустая.
