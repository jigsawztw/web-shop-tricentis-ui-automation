### TC-SMOKE-11: Открытие формы регистрации.

**Описание:** Проверка открытия формы регистрации пользователя.  
**Предусловие:** Нет.  
**Шаги:**

1. Щелкнуть по кнопке "Register" в хэдере сайта.

**Expected result:**

- Открылась форма регистрации, заголовок Register виден на странице, на форме отображаются элементы:

1. Радиобаттоны "Gender" (Male/Female).
2. Инпуты "First name", "Last name", "Email", "Password", "Confirm Password" доступны для ввода.
3. Кнопка "Register" доступна для нажатия.

### TC-SMOKE-12: Открытие формы авторизации.

**Описание:** Проверка открытия формы авторизации пользователя.  
**Предусловие:** Нет.  
**Шаги:**

1. Щелкнуть по кнопке "Log in" в хэдере сайта.

**Expected result:**

- Открылась форма авторизации, на странице отображается приветственное сообщение
  "Welcome, Please Sign In!", на форме отображаются элементы:

1. Инпуты Email, Password доступны для ввода.
2. Чекбокс "Remember me?" доступен для выбора.
3. Ссылка "Forgot password?" доступна для нажатия.
4. Кнопка "Log in" доступна для нажатия.
