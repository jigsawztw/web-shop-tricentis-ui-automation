# Regression Tests

## Корзина
### TC-REGRESSION-01: Добавление товара в корзину авторизованным пользователем
**Описание:** Проверка добавления товара в корзину.  
**Предусловие:** Пользователь авторизован в системе.  
**Шаги:**
1. Выбрать любую категорию, например "Computers".
2. Выбрать любой из товаров категории и нажать кнопку "add to cart".
3. Открыть корзину.  

**Expected result:**  
- Корзина открыта, добавленный товар из категории книги отображается в корзине.  



### TC-REGRESSION-02: Удаление товара из корзины авторизованным пользователем
**Описание:** Проверка удаления товара из корзины.  
**Предусловие:** Пользователь авторизован в системе.  
**Шаги:**
1. Выбрать любую категорию, например "Books".
2. Выбрать любой из товаров категории и нажать кнопку "add to cart".
3. Открыть корзину.
4. Нажать кнопку "remove from cart".  

**Expected result:**  
- Корзина открыта, товар удален из корзины, корзина пустая.

## Регистрация
### TC-REGRESSION-04: Успешная регистрация пользователя 
**Описание:** Регистрация нового пользователя.  
**Предусловие:**   Перейти на главную страницу.  
**Шаги:**
1. Кликнуть по кнопке Register.
2. Выбрать один из радиобаттонов для поля Gender. 
3. Заполнить поле Имя.
4. Заполнить поле Фамилия.
5. Заполнить поле Email.
6. Заполнить поле пароль.
7. Заполнить поле подтверждения пароля.
8. Нажать кнопку Register.

**Expected result:**  
- Регистрация завершилась успешно, получено сообщение Your registration completed, кнопка Logout доступна для нажатия. Вверху экрана отображается email зарегистрированного пользователя.
